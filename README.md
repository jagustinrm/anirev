# PáginaPrueba — Next.js app

Este es el README principal para la aplicación Next.js ubicada en `next-app/`.

Instrucciones rápidas:

1) Instalar dependencias

```powershell
cd next-app
npm install
```

2) Desarrollo

```powershell
npm run dev
```

3) Build / producción

```powershell
npm run build
npm run start
```

4) Configurar variables de entorno

- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY (server-only)
- NEXT_PUBLIC_APP_URL

Ver `DEPLOY.md` para detalles de despliegue en Vercel.
